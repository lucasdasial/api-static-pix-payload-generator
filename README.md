### Caso queira rodar na sua máquina

Inicia o servidor de desenvolvimento

```bash
yarn dev
```

Compila o codigo, gerando os arquivos para produção

```bash
yarn build
```

Inicia o servidor com os arquivos de produção

```bash
yarn start
```
